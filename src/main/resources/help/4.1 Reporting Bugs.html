Reporting Bugs

If you find a problem with AstroJournal please try the following steps:

  1. Check that you are running the latest version of AstroJournal. You can see the version you are running on the main window title and you can check the AstroJournal website (http://pdp10.github.io/AstroJournal/) to see what the latest version is
  2. Report a bug using AstroJournal online bug reporting and tracking system at https://github.com/pdp10/AstroJournal/issues.

You can also suggest new functionalities you would like to see in AstroJournal by filing them as a bug with a severity of 'enhancement'.
